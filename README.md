# Estrutura de Dados
- Desenvolve o conceito de tipos abstratos de dados e as estruturas de dados básicas: listas, pilhas, filas e conjuntos.
- Técnicas de representação dessas estruturas, sequencial e encadeada, e aplicações das mesmas.
- Analisa a representação, manipulação e aplicação de árvores, árvores binárias, árvores binárias de busca, heaps, árvores balanceadas, árvores multidirecionais, tabelas hash e mapas.

Matéria do 3° semestre de Ciência da Computação da UNISATOS (Universidade Católica de Santos).
